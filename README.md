

# FlexOrder

一个简易的线上订餐WEB系统，可以用于课程作业或者是毕业设计。

采用MVC分层架构进行开发，提供软件工程UML类图和各种用例图

## 技术栈

后端部分：Java,Servlet,Jdbc,Mysql

前端部分：JSP,BootStrap,JavaScript,CSS,HTML

## 开发工具

Eclipse,Tomcat
